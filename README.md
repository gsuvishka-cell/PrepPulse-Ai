# PrepPulse AI 

PrepPulse AI is an intelligent interview readiness analyzer designed to help students evaluate their preparation before facing real recruiters.

## Problem Statement
Many students attend interviews without knowing their weak areas. PrepPulse AI solves this problem by providing a quick readiness assessment in less than 2 minutes.

## Features
- Technical Skill Analysis
- Resume Evaluation
- Communication Assessment
- Portfolio/GitHub Review
- Instant Readiness Score
- Personalized Improvement Suggestions

## Readiness Levels
- Beginner
- Intermediate
- Job Ready
- Interview Pro

## Tech Stack
- Python
- AI-based Scoring System
- GitHub
